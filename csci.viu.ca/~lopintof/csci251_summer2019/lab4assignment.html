Lab 4 Assignment

  • Create (or reuse) two virtual tiny core machines
  • Name one Client and the other Server
  • Assign IP addresses to them
  • Modify the hosts files so that Client has an entry to reach Server and Server has an entry to reach Client
  • Install a web server in Server and a Web client in Client
  • Make your changes persistent. (Hint: Check References on my website.)
  • Create s simple web page and put it on the server
  • Reboot both computers and demonstrate that you can retrieve your web page from the server using a browser on the client
