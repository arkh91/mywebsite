Lab 3 Assignment

  1. Create two virtual machines running TinyCore and able to compile C programs. Reuse existing machines if you want.
  2. Use bootcodes to persist home and opt and to give the machine a hostname.
  3. Name one host Client and the other Server.
  4. Determine the IP addresses of both machines.
  5. Ping the client from the server.
  6. Ping the server from the client.
  7. Search the Web for an example of socket programming in C.
  8. Implement a client and a server on the same machine and show that they can exchange information.
  9. Repeat the demonstration with the client program running on Client and the server program running on Server

Note: You will need to setup Host Only Adaptors

  1. Select File/Preferences in the Virtual Box Manager
  2. Select Network
  3. On the network menu select the Host-only Network tab
  4. Click the Add button (top-right) to create a Host-only network.
  5. Your adaptor should have an IP address that starts with 192.
