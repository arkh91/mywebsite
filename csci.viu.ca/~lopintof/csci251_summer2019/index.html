The expected learning outcomes from this course are to:

  • Know how to install and configure a Linux system
  • Know how to add hardware to the system
  • Know how to install and maintain software
  • Have a basic knowledge of how to build a desktop system
  • Have a basic knowledge of how the Internet works
We will use virtual machine technology to manaage a working Linux system with superuser addess.
Topics include:
  • process management
  • disk management
  • system management
  • network management
  • kernel management
  • backup management
