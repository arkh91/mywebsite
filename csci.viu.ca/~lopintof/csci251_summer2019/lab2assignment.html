Lab 2 Assignment
  1. Create a virtual machine (VM) using VirtualBox
  2. Install Tiny Core Linux (Frugal Install) in your VM
  3. Install VirtualBox Guest Additions on your machine
  4. Create a copy of the machine and use it for the rest of the lab
  5. Add a second disk (10 GB)
  6. Create two partitions on the second disk (4 and 6 GB)
  7. Persist /home on the 6GB partition and /opt on the other one
  8. Install GCC on the machine
  9. Write and run a C or C++ program that displays Hello World!
