The expected learning outcomes from this course are:

  • Be able to use all the basic HTML elements
  • Gain hands-on experience with some of the new HTML 5 elements including
    • Video
    • Audio
    • Graphics
  • Be able to style text using CSS
  • Be able to position elements on the screen
  • Learn how to develop pages that look good on phones as well as computer screens
  • Learn basic Javascript to change the contents and appearance of a web page

Lab assignments

Lab 1
Create a web page using the links in the initial version of this site. See Version 0 in the Archive.
Lab 2
Create a web page that uses CSS to style at least two different elements. Use external and internal style sheets as well as styling by tag attributes.
Experiment with including stylesheets in different orders. Use the page you create to explain what you did and what you saw.
Lab 3
Create a page that's linked to your main page. Use the new page to experiment with:
  • The box model
  • The selectors discussed in class
Lab 4
Experiment with positioning.
  • Create a page called Lab 4 and link to it from your home page
  • Style some text using <span> and <div>
  • Position divs using fixed, relative, absolute, and sticky positioning
Lab 5
Experiment with Menus. The goal of this lab is to give you experience with styled menus. Refer to CSS Navigation Bar in the References section and do the following:
  1. Spend your two hours creating vertical and horizontal menus
  2. Use the Hover Pseudo Selector to change the appearance of menu items. See References section>
  3. Submit a single HTML page showing me how your can style menus. They don't have to have to go anywhere.
  4. Make sure your Name and ID are displayed at the top of your page.
  5. The page is a report. Tell me what you tried, accomplished, and learned. Your grade will depend on what you do and how well you explain yourself.
Lab 6
Experiment with Keyframes. The goal of this lab is to give you experience using CSS3 animations.
  1. Spend your time developing a page with animation effects.
  2. Refer to the references section for "Fun with Keyframes"
  3. Explore more at w3 Schools / CSS Animations
  4. Follow the instructions for Lab 5, steps 4 and 5, and have some fun with it!

Lab grades will be assigned as follows:

  • 0 - Nothing submitted. 0%
  • 1 - Assignment satisfied by cut-and-paste from web post. 60%
  • 2 - Assignment satisfied with evidence of experimentation. Report explains experimentation. 85%
  • 3 - Report shows lessons learned and insights gained. 100%
