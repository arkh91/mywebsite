CSCI 330 Fall 2008: Trees in Scheme


October 23, 2008

Dave Wessels' Scheme Scheme page Dave's Quick Reference.

The following file is provided. Cut and paste it into a file.

  1. The Scheme function "apply-to-all", which we covered in class, is in the file provided. Use it to square every integer in a list.

  2. Also provided are functions allowing you to define trees, with left and right children and data in the node.

    What does the node function return? Examine the code in the file. Explore by defining, in the interpreter, a tree t2 with one left child and no right child. What is returned when you type (t2 0)? (t2 1)? (t2 2)? (t2 3)?

  3. Write a function called size which takes a tree and returns the number of nodes in the tree. Test your size function.

  4. Write a search function that takes a key value and a tree in BST order and returns the node containing a key if the sought-after value is a key in the tree; () otherwise. Test is by writing a function called found that returns true if the key is in the tree, false otherwise.

  5. Write a function called traverse that traverses the tree and outputs the data values in inorder.
