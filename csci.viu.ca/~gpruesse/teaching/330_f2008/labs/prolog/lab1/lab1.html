Specifying ranges of values in Prolog

2. Here is a simple Prolog program to generate the ways of making change for a dollar. 

change([H,Q,D,N,P]) :-
	member(H,[0,1,2]),          /* half-dollar */
	member(Q,[0,1,2,3,4]),      /* quarters */
	...
	S is 50*H + ...
	S =< 100,
	P is 100-S.
Finish off the code so that you can query various ways of making change for a dollar.
