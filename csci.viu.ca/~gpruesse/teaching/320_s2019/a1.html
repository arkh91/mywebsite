Assignment 1

Due: 2:30 Wed Jan 23 2019

Objective
To demonstrate understanding of Regular Languages (FAs and REs).

Task

    1. (10 marks) Design a DFA that recognizes the language
      L = { w ∈ {a,b}*: |w| ≥ 2 and the 2-letter prefix of w is the reverse of the 2-letter suffix of w }.
      For full marks, it must have as few states as possible and be correct. How many states does it have?
    2. (2 marks) Suppose you were asked to give a DFA for the language
      L ∩ {w: |w| is even}
      What is the minimum number of states in such a DFA? Explain your answer. (An explanation will give the meaning of each state.)
    3. (1 marks) Give a RE for the language L of part 1(b).
  2. (10 marks) Use the construction given in class to devise a NFA that accepts the language (ab* + ac)*. Then give a smaller NFA that accepts the same language.
  3. (10 marks) Use the construction given in class to devise the DFA for the NFA with the following transition function. Give the DFA as a diagram. The alphabet is {a,b,c}, and the set of states is {1,2,3}. δ = {(1,a,{1}), (1,ε,{2}),(1,b,{2}),(2,a,{2,3}),(2,c,{3}),(3,ε,{1}), (3,b,{1}),(3,c,{3})}. State 1 is the start state and is the only accept state.
  4. (10 marks) Use the construction given in class to devise the RE for the language of the FA with the following transition function. δ = { (1,a,1), (1,b,2), (2,a,2), (2,b,1)}.
    State 1 is the start state and is the only accept state.
