Assignment 3


Due Wednesday March 6 (changed from Tue March 5), 2019 2:30 pm. Do not send electronic submissions without permission.

Objective
To problem-solve within the particular constraints of limited computational models: Push-Down Automata and Context-Free Grammars.
To develop understanding of the models and to create algorithms.
To foster understanding of set cardinality, countable and uncountable sets, and how to prove cardinality claims about sets.

Task

  1. (15 marks) Give a natural PDA and a CFG for the language L, where L={w ∈ {0,1}*: |w| is even and the first half of w is not the same as the second half of w}. For example, aabaab is not in L, but aababb is in L.
  2. (10 marks) Prove that the class CFL is closed under string reversal.
  3. (10 marks) Give a bottom-up parser for the language generated by the following grammar. Use the construction.
    S→ aSdd | aSd | X
    X → bXc | bbXc | ε
  4. (5 marks) What language is generated by the grammar given in the previous question? For full marks, give a formal but simple definition.
  5. (10 marks) Give a top-down parser for the language generated by the following grammar. Use the construction.
    S→ AXD
    X → bX | ε
    A → aAb | ε
    D → bDc | ε
  6. (10 marks) Let W = { (i,j,k): i,j,k ∈ N }, where N is the set {1,2,3, ...}. Prove that W is countably infinite.
  7. (10 marks) Let Z = { s: s is an infinite sequence of letters from the set {a,b,c} }. Prove that Z is uncountably infinite.
